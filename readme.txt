Insert YouTube videos using either the FuTube UI or standard YouTube Chrome.

== Description ==

Once you have installed and activated the plug-in you can add Fubra Videos to your blog post using the following short tag:

[futube video="gE1KGHUb9zw"]

A fully customised player can be included by defining any of the available options:

[futube video="gE1KGHUb9zw" align="left" author="by Fubra Limited" bgcolor="#FFFFFF" color="#336699" hd="true" height="300" image="/path/to/image.jpg" title="Example Video" width="400"]

To included videos with the standard user interface use:

[youtube video="gE1KGHUb9zw"]
